How this dashboard works

  1. Each zone shows real ecosystem state pulled from the canonical markdown files in this repo.
  2. KPI inputs at the top persist to your browser's localStorage — they don't leave your machine.
  3. Every ▶ button generates a prompt for a Claude Code subagent defined in .claude/agents/.
  4. "Full ecosystem audit" dispatches the ops-orchestrator, which runs the three specialist agents in parallel and synthesizes one report — agents running agents.
  5. Paste the generated prompt into Claude Code. The right subagent is invoked automatically.
